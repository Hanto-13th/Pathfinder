### ‚ú® Features

1. Visualization of the algo on the CLI in ASCII.

2. Possibilities for the user to choose the size of the matrice.

3. Possibilities for the user to choose the start and the target nodes.

4. The matrice is composed with N x N nodes

5. The start node is represented with 'A', the target node with 'B' and the other node with '#'

6. The algo displays the number of nodes from the started node to the target
